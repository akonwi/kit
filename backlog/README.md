# Backlog

This directory tracks outstanding work for the standalone `pi-kit` app.

## Conventions

- `[ ]` not started
- `[x]` done

## Files

- `roadmap.md` — phase-oriented view of the remaining work
- `refinements.md` — product and architectural refinements deferred until later

When work is completed, mark it done in the relevant backlog file rather than letting it drift in chat history.
